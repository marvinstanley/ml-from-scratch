kNN module¶

This module implements k nearest neigbors algorithm

class ml_scratch.neighbors.KNNClassifier(*args, **kwargs)[source]¶

Nearest Neighbor for Categorical target values. Counts the highest occuring target label from the near neighbors.

Note

It currently only accepts numpy arrays. Be sure to convert your data to numpy arrays

from ml_scratch.knn import KNNRegressor
import numpy as np

model = KNNRegressor(3)

A_x = np.array([[1, 2, 3, 4],
            [5, 5, 6, 7],
            [1, 4, 2, 3],
            [3, 1, 3, 7],
            [2, 8, 1, 9],
            [1, 5, 9, 7]])

# A_y = ['A', 'A', 'A', 'B', 'B', 'B']
A_y = [1.0, 2.5, 0.6, -1.0, 5.0, 3.0]

B_x = np.array([[5, 3, 2, 1],
            [8, 1, 2, 2]])

model.fit(A_x, A_y)
print(model.predict(B_x))

class ml_scratch.neighbors.KNNRegressor(*args, **kwargs)[source]¶

Nearest Neighbor for continous target values. Uses interpolation/mean to find the nearest value from the near neighbors.

Note

It currently only accepts numpy arrays. Be sure to convert your data to numpy arrays

# #############################################################################
# Generate sample data
import numpy as np
import matplotlib.pyplot as plt
from ml_scratch.knn import KNNRegressor

np.random.seed(0)
X = np.sort(5 * np.random.rand(40, 1), axis=0)
T = np.linspace(0, 5, 500)[:, np.newaxis]
y = np.sin(X).ravel()

# Add noise to targets
y[::5] += 1 * (0.5 - np.random.rand(8))

# #############################################################################
# Fit regression model
n_neighbors = 10


knn = KNNRegressor(n_neighbors)
y_ = knn.fit(X, y).predict(T)

plt.scatter(X, y, color='darkorange', label='data')
plt.plot(T, y_, color='navy', label='prediction')

plt.legend()
plt.title("KNeighborsRegressor")

plt.show()
